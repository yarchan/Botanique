# Тестовое задание

клиентская и серверная часть

## Установка

1. Клонируйте репозиторий: `git clone https://github.com/yarchan/Botanique.git`
2. Перейдите в каталог проекта
3. Запустить команду docker build -t botanique .  
4. Запустите команду docker run -p 3000:3000 botanique
5. Перейдите в браузер и откройте http://localhost:3000/
